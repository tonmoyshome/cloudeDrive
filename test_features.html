CloudeDrive New Features Test

✅ Features Implemented

  • ✓ Rename Files & Folders: Users with full access can rename files and folders
  • ✓ Multi-Select Delete: Select multiple files/folders and delete them at once
  • ✓ Home Folder Access: Users can create/upload in their username folder and subfolders
  • ✓ Shared Folder Access: Users can create/upload in shared folders with upload permission
  • ✓ Folder Sharing: Users can share folders they created
  • ✓ Admin Full Access: Admin always has full access to all folders

🎯 How to Test

  1. Login as Admin: Go to main page and login with admin credentials
  2. Create Test Users: Add new users through admin dashboard
  3. Test Rename: Click the edit (pencil) icon next to any file/folder you own
  4. Test Multi-Select: Use checkboxes to select multiple items, then use the action buttons
  5. Test Permissions:
    • Try uploading to your username folder - should work
    • Try uploading to someone else's folder - should be denied unless shared
    • Share a folder with another user and test their access
  6. Test Admin Access: Login as admin and verify you can access all folders

🔐 Permission Rules

  • ℹ Admin: Full access to everything, always
  • ℹ File Owner: Can rename, delete, and manage their own files
  • ℹ Folder Owner: Can rename, delete, share, and manage their own folders
  • ℹ Home Folder: Users can create/upload in folders matching their username
  • ℹ Shared Access: Based on granted permissions (view, download, upload, delete)
  • ℹ Root Level: Only admin can create folders at root level

🔗 API Endpoints Added

  • /api/rename_file.php - Rename files
  • /api/rename_folder.php - Rename folders
  • /api/delete_multiple_files.php - Delete multiple files
  • /api/delete_multiple_folders.php - Delete multiple folders
